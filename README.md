## How to run Plunker for components locally

- `npm install -g serve@6.5.3`
- In skyux2 repo
  - `npm run build`
  - `serve . --cors --ssl --cache 0`
  - Open `https://localhost:5000` and approve SSL cert
- In skyux2-docs repo
  - `skyux serve`
