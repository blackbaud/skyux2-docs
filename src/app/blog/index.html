Design update — May 2018

SKY UX releases 2.12.0-2.12.1

Recently completed

Select field

Ported from SKY UX 1. Provides a control for when users need to select one or many items from a list but don't know the exact value they are looking for.

Corner and shadow CSS classes

We added CSS classes for shadows and rounded corners. Previously these were only available within components as a SCSS mixin and SCSS variable, respectively.

In the works

Infinite scroll

The infinite scroll component will be ported from SKY UX 1.

List toolbar

The list toolbar will be updated to follow new UX guidelines.

Toast

The toast component will be ported from SKY UX 1.

Spacing CSS classes

We will add CSS classes to cover common spacing needs.

Design update — March 2018

SKY UX releases 2.10.0-2.11.0

Recently completed

Lookup

The lookup component combines typeahead search functionality with the option to select one or multiple items for data entry.

Flyout

Added the ability for the user to change the width of the flyout panel.

Alert, Label, and Status indicator

Updated the styling of these three components to improve consistency and accessibility.

UX guidelines

Updated the UX guidelines for error handling, form design, and user assistance.

In the works

Select field

The select field component will be ported from SKY UX 1.

Infinite scroll

The infinite scroll component will be ported from SKY UX 1.

List toolbar

The list toolbar will be updated to follow new UX guidelines.

Design update — February 2018

SKY UX releases 2.8.0-2.9.0

Recently completed

Autocomplete

The autocomplete component provides a typeahead search field. This is part of the background work to support the upcoming lookup component.

Flyout

The flyout component provides a content container that slides in from the right side of the page to display supplemental information without requiring users to navigate away.

In the works

Lookup

The lookup component will combine typeahead search functionality with the option to select one or multiple items for data entry.

Select field

The select field component will be ported from SKY UX 1.

List toolbar

The list toolbar will be updated to follow new UX guidelines.

Design guidelines
