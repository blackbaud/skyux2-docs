Welcome to Blackbaud's user experience framework!

SKY UX brings a consistent, cohesive experience to Blackbaud products and allows third-party developers to provide the same consistent experience in their customizations and applications. SKY UX provides guidelines and tooling for the entire application lifecycle.

 
Get started SKY UX overview
 

Open source

SKY UX is proudly open sourced and hosted on GitHub. We highly encourage contributions from all SKY UX users. Learn more.

Design patterns

The core of SKY UX is our design patterns. They empower users, focus on their workflow, and provide a clean, cohesive interface. Learn more.

   

Angular components

We implement our design patterns as Angular components that allow developers to quickly create applications with a common user experience. Components are grouped into modules to improve performance. Learn more

To learn about our AngularJS (1.x) implementation, visit the SKY UX 1 documentation.

SDK

SKY UX provides additional tooling in the form of several SDK and CLI libraries. These ensure rapid application development, as well as reliable testing and hosting of our front-end applications. Learn more.

   

Docs as Code

The Stache component library provides open source components built on SKY UX to generate consistent, user-focused technical documentation websites with the same tools that you use to write code. Learn more.
