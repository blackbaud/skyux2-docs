Welcome to Blackbaud's user experience framework!

SKY UX brings a consistent, cohesive experience to Blackbaud products and allows third-party developers to provide the same consistent experience in their customizations and applications. SKY UX provides guidelines and tooling for the entire application lifecycle.

 
Get started
 

Open source

SKY UX is proudly open sourced and hosted on GitHub. We highly encourage contributions from all SKY UX users and just ask that they follow our existing coding conventions. Learn more.

Design patterns

The core of SKY UX is our design patterns, which empower users, focus on their workflow, and allow us to maintain a clean, cohesive interface. Learn more.

   

Angular components

We implement our design patterns as Angular components that allow developers to quickly create applications with a common user experience. Components are grouped into modules to improve performance. Learn more.

SDK

SKY UX provides additional tooling in the form of several SDK and CLI libraries. Learn more.
