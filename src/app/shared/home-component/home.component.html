A user experience framework to build web applications that leverage common Blackbaud design patterns.
Welcome to Blackbaud's user experience framework!

SKY UX brings a consistent, cohesive experience to Blackbaud products and allows third-party developers to provide the same consistent experience in their customizations and applications. SKY UX provides guidelines and tooling for the entire application lifecycle.

 
Get Started SKY UX Overview
 

Open Source

SKY UX is proudly open sourced and hosted on GitHub. Check out our contributing page for information on our guidelines for contributing, past contributors, and a list of repositories available for contribution.

SKY UX Design Patterns

Driven by design patterns. The core of SKY UX is defined by our design patterns, ensuring a consistent, cohesive experience. Learn more.

   

Angular Components

Our design patterns are implemented as Angular Components. This allows developers to quickly create applications which provide a common user experience. These components are grouped into several modules for best performance. Learn more.

SDK

Additional tooling is providing in the form of several SDK and CLI libraries. These ensure rapid application development, as well as reliable testing and hosting of our front-end applications. Learn more.
