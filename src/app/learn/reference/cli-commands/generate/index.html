skyux generate creates template items for your SKY UX project. This command requires two arguments. First, it requires a type argument to specify the type of item to create. Currently, component is the only available option for the type argument. And second, the command requires a name argument to provide a name for the template items.

Arguments

  • Specify the type of template item to create. Currently, component is the only available option for this argument, and it generates the TypeScript, HTML, CSS, and spec files for a new component.

  • Provide a name for the template items to create. You do not need to include file extension information. You can precede the name with an optional subfolder to create a subfolder in the src/app folder. If you do not specify a subfolder, the generate command places items directly in the src/app folder.
