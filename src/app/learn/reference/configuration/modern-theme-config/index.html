Configure the modern theme

  1. From the command prompt, cd into the root directory for your SPA.

  2. Run skyux upgrade to update the SPA's dependencies with the versions of the SKY UX libraries that support theming.

  3. In your SPA's skyuxconfig.json, add the theming property to the app section.

    4. "app": { "theming": { "supportedThemes": ["modern"], "theme": "modern" } }

To verify the theme update, run skyux serve. The page's background color should be an almost white shade of gray, and the default font size should be 16px.

Themed components

After you configure your SPA for the modern theme, you can use modern themed components by installing the libraries that support them as they become available. Currently, the modern theme only contains base styles, such as font and background colors, and CSS classes that enable use of the box component.

Box component

The box component is not an Angular component but a combination of CSS classes that you can apply to elements such as a div. This component is a common element in modern theme designs.

To add a rounded corners and a dropshadow to your element, create an element and add the sky-box and sky-elevation-1 CSS classes to it.

Example: <div class="sky-box sky-elevation-1">I'm in a box</div>
