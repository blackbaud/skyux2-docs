SKY UX Builder allows you to track parameters in the URL and expose them through your app using the SkyAppConfig class. Below is an example:

import { Component } from '@angular/core'; import { SkyAppConfig } from '@blackbaud/skyux-builder/runtime'; @Component({ selector: 'my-home', templateUrl: './home.component.html' }) export class HomeComponent { public envId: string; constructor(skyAppConfig: SkyAppConfig) { this.envId = skyAppConfig.params.get('envid'); } }

For more information about envid and other querystring parameters, see the params configuration property and how to specify default svcid or envid values for local development.
