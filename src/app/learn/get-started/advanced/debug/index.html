While you can debug SKY UX code in your browser using developer tools, you can also debug the code in Visual Studio Code. That option just requires a few additional steps.

Debug in Visual Studio Code
  1. Follow the procedures in the VS Code - Debugger for Chrome repo.
  2. From the terminal, launch Chrome with --remote-debugging-port=9222.
  3. In VS Code, run skyux serve.
  4. In VS Code, run the Attach to Chrome or Attach to Edge task in VS Code.
