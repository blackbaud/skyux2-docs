Although we recommend debugging with Firefox or Chrome, you can also use Visual Studio Code to debug your code.

Debug in Visual Studio Code
  1. Follow the procedures listed in the VS Code Debugger for Chrome README.
  2. Close any previously opened instances of Chrome and then launch Chrome with --remote-debugging-port=9222.
  3. In VS Code, run skyux serve.
  4. Run the Attach to Chrome or Attach to Edge task in VS Code.
