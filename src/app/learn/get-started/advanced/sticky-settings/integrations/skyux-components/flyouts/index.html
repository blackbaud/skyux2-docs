To save user configuration settings in flyouts, add the settingsKey property to the config parameter of the open(component, config?) property that the flyout service uses to open the flyout. The settingsKey must be unique to the component.

The UI Config Service preserves the width of flyouts.

public openFlyoutWithCutsomWidth(): void { const flyoutConfig: SkyFlyoutConfig = { defaultWidth: 350, maxWidth: 500, minWidth: 200, settingsKey: "my-flyout" }; this.flyout = this.flyoutService.open(SkyFlyoutDemoInternalSimpleComponent, flyoutConfig); }

That's it! The UI Config Service now saves the width of the flyout when users close it and restores the width when users reopen it.
