After you update the SKY UX CLI with the latest version, you can build and publish your SPA.

Blackbaud developers should create repos and provision applications through the SKY UX App Management Portal. Documentation on the SKY UX App Management Portal is available on the Engineering System website. This site is for Blackbaud internal use only and is not available to external developers.

  1. From the command prompt, cd into the application's directory.

  2. Enter skyux build to build the SPA into the dist/ folder.

  3. Copy the contents of the dist/ folder into a directory such as my-spa/, wheremy-spa is the name of your SPA.

  4. Store the my-spa/ directory on a web server.

  5. In a browser, enter https://example.com/my-spa to view your SPA.
