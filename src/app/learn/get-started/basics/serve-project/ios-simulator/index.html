To serve your SPA to an iOS simulator and debug it, follow these instructions.

  1. Download and install XCode. This install can take a long time.

  2. Open XCode and select XCode in the toolbar.

  3. Navigate to Open Developer Tool and select Simulator. This opens the iOS simulator to simulate a full iOS device.

  4. Select the simulated power button to turn on the simulator.

  5. Set up the simulated device to allow local development.

    • On your iOS device, navigate to Settings, Safari and turn off Prevent Cross-Site Tracking.

    • On your Mac, open Keychain Access to extract the SKY UX certificate from your machine for use in iOS.

    • Select the SKYUX-Developer-CA certificate and then navigate to File, Export Items.

    • Save the certificate to your preferred location.

    • Drag the .cer file to the simulator. No prompt appears.

  6. Run skyux serve --browser safari to serve your SPA to Safari, and then use the link provided in the command line to view your SPA.

  7. To allow console logging and general debugging, follow these steps:

    • On your Mac, open Safari and navigate in the toolbar to Safari, Preferences, Advanced and enable Show Develop menu in menu bar.

    • Open http://preview.mobify.com in the iOS simulator.

    • Paste the URL for your locally served SPA into the Site URL box and select Preview.

    • On your Mac, Select Safari and navigate in the toolbar to Safari, Develop.

    • The dropdown includes a Simulator entry. Place the cursor on it, and then select the URL that matches your locally served SPA. From here, you can debug locally served changes in real time.
