During local development with SKY UX, local files are loaded into SKY UX Host. To load your local single-page application into SKY UX Host, you must install the skyux-ca.crt root certificate. Follow the installation instructions appropriate for your machine.

Automated Installation For Linux, Mac, and Windows skyux certs trust

Visit the skyux certs documentation to learn about other available options.

Manual Installation for Mac
  1. skyux certs generate

  2. Open the Keychain Access application.

  3. Under the Keychains list on the left, select login.

  4. Under the Category list, select Certificates.

  5. Open the .skyux/certs folder in your home directory and drag into the list of certificates.

  6. Double-click the new SKY UX item in the list to open the certificate's info window.

  7. Expand the Trust section near the top of the info window, and select the Always Trust option for Secure Sockets Layer (SSL).

  8. Close the info window.

Manual Installation for iOS Simulator on Mac
  1. skyux certs generate

  2. Open the .skyux/certs folder in your home directory.

  3. Drag and drop the skyux-server-cert.pem certificate to your simulator.

  4. In the prompt that appears, select Allow to download the profile and then select Close.

  5. Open the Settings app and navigate to General > Profile > SKY UX.

  6. Select Install on each page (3 times total) to install the certificate.

  7. Select Done to exit.

  8. In Settings > General > About, select Certificate Trust Settings.

  9. Select Enable Full Trust For Root Certificates for the SKY UX certificate and continue when prompted.

Manual Installation for Windows
  1. skyux certs generate

  2. Open the .skyux/certs folder in your home directory.

  3. Open the skyux-server-cert.pem file.

  4. On the security warning prompt, select Open.

  5. On the Certificate screen, select Install Certificate.

  6. On the Certificate Import Wizard, select Next.

  7. Select Place all certificates in the following store, and then select Browse.

  8. On the Select Certificate Store screen, select Trusted Root Certification Authorities, and then select OK.

  9. On the Certificate Import Wizard, select Next.

  10. Select Finish.

  11. On the security warning prompt, select Yes.

  12. On the Certificate Import Wizard confirmation prompt, select OK.

  13. After you return to the Certificate screen, select OK.

Manual Installation for Firefox
  1. skyux certs generate

  2. Enter "about:preferences#privacy" in the Firefox address bar.

  3. Near the bottom of the page, select View Certificates.

  4. Under Servers on the Certificate Manager, select Import.

  5. Open the skyux-server-cert.pem from the .skyux/certs folder in your home directory.

  6. On the options dialog, select Trust this CA to identify websites and then select OK.

  7. On the Certificate Manager, select OK.

Manual Installation for Linux
  1. skyux certs generate

  2. Copy the skyux-server-cert.pem from the .skyux/certs folder in your home directory to /usr/local/share/ca-certificates.

  3. From the command terminal, run sudo update-ca-certificates.

    When you use products that use their own certificate stores, you also need to add the certificate to those certificate stores. For example, follow the instructions above to install the certificate for Firefox, import the certificate in Google Chrome, or add the certificate to the Java certificates store.

  4. To confirm that the certificate installed successfully, verify that it is included in the concatenated list of trusted certificates in /etc/ssl/certs/ca-certificates.crt.
