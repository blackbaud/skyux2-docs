For local development with SKY UX, you must install the skyux-server-cert.pem certificate that enables your SPA to load local files into SKY UX Host.

Automated installation for Linux, Mac, and Windows

To add the skyux-server-cert.pem certificate to your machine's list of trusted certificates and validate it, run the following command:

skyux certs install

The certificate is required to securely run skyux serve, skyux e2e, and skyux build --serve. For more information, see the skyux certs documentation.

Additional steps for iOS Simulator on Mac

  1. Open the Settings app, and navigate to General > About.

  2. Select Certificate Trust Settings.

  3. Select Enable Full Trust For Root Certificates for the SKY UX certificate and continue when prompted.

Additional steps for Firefox

  1. Enter "about:preferences#privacy" in the Firefox address bar.

  2. Near the bottom of the page, select View Certificates.

  3. Under Servers on the Certificate Manager, select Import.

  4. Open the skyux-server-cert.pem from the .skyux/certs folder in your home directory.

  5. On the options dialog, select Trust this CA to identify websites and then select OK.

  6. On the Certificate Manager, select OK.
