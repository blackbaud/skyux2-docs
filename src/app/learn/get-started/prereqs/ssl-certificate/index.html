During local development with SKY UX, local files are loaded into SKY UX Host. To load your local single-page application into SKY UX Host, you must install the skyux-ca.crt root certificate. Follow the installation instructions appropriate for your machine.

Installation for Mac

  1. Download the skyux-ca.crt. To download, right-click the link and save it to your computer.

  2. Open the Keychain Access application.

  3. Under the Keychains list on the left, select login.

  4. Under the Category list, select Certificates.

  5. Drag skyux-ca.crt into the list of certificates.

  6. Double-click the new SKY UX item in the list to open the certificate's info window.

  7. Expand the Trust section near the top of the info window, and select the Always Trust option for Secure Sockets Layer (SSL).

  8. Close the info window.

Installation for iOS Simulator on Mac

  1. Download the skyux-ca.crt to your Mac. To download, right-click the link and save it to your computer.

  2. Drag and drop the certificate to your simulator.

  3. In the prompt that appears, select Allow to open a settings page.

  4. Select Install on each page (3 times total) to install the certificate.

  5. Select Done to exit.

  6. In Settings > General > About, select Certificate Trust Settings.

  7. Select Enable Full Trust For Root Certificates for the SKY UX certificate and continue when prompted.

Installation for Windows

  1. Download the skyux-ca.crt. To download, right-click the link and save it to your computer.

  2. Open the file.

  3. On the security warning prompt, select Open.

  4. On the Certificate screen, select Install Certificate.

  5. On the Certificate Import Wizard, select Next.

  6. Select Place all certificates in the following store, and then select Browse.

  7. On the Select Certificate Store screen, select Trusted Root Certification Authorities, and then select OK.

  8. On the Certificate Import Wizard, select Next.

  9. Select Finish.

  10. On the security warning prompt, select Yes.

  11. On the Certificate Import Wizard confirmation prompt, select OK.

  12. After you return to the Certificate screen, select OK.

Installation for Firefox

  1. Download the skyux-ca.crt. To download, right-click the link and save it to your computer.

  2. Enter "about:preferences#privacy" in the Firefox address bar.

  3. Near the bottom of the page, select View Certificates.

  4. Under Authorities on the Certificate Manager, select Import.

  5. Open the skyux-ca.crt.

  6. On the options dialog, select Trust this CA to identify websites and then select OK.

  7. On the Certificate Manager, select OK.

Installation for Linux

  1. Download the skyux-ca.crt. To download, right-click the link and save it to your computer.

  2. Move the skyux-ca.crt to /usr/local/share/ca-certificates.

  3. From the command terminal, run sudo update-ca-certificates.

    When you use products that use their own certificate stores, you also need to add the certificate to those certificate stores. For example, follow the instructions above to install the certificate for Firefox, import the certificate in Google Chrome, or add the certificate to the Java certificates store.

  4. To confirm that the certificate installed successfully, verify that it is included in the concatenated list of trusted certificates in /etc/ssl/certs/ca-certificates.crt.
