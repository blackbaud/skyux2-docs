The text expand component truncates long blocks of text with an ellipsis and a link to expand the full text. When users click the link, the component expands the full text inline unless it exceeds limits on text characters or newline characters. If the text exceeds those limits, then the component expands the full text in a modal view instead. The component does not truncate text when it is shorter than a specified threshold. However, it always truncates text that includes newline characters, and by default, it removes newline characters from truncated text.

Specifies the text to truncate. This property accepts string values. Specifies a title to display when the component expands the full text in a modal. This property accepts string values. Specifies the maximum number of text characters to display inline when users click the link to expand the full text. If the text exceeds this limit, then the component expands the full text in a modal instead. This property accepts number values. Specifies the maximum number of newline characters to display inline when users click the link to expand the full text. If the text exceeds this limit, then the component expands the full text in a modal view instead. This property accepts number values. Specifies the number of text characters to display before truncating the text. To avoid truncating text in the middle of a word, the component looks for a space in the 10 characters before the last character. This property accepts number values. Indicates whether to replace newline characters in truncated text with spaces. This property accepts boolean values.
