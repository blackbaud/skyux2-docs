The copy to clipboard component creates a button that enables users to copy content to their clipboard.

To use the copy to clipboard component, set it up as a dependency in your SKY UX SPA. First, install the package as part of your SPA dependencies: npm install @blackbaud/skyux-lib-code-clipboard --save. Then add SkyClipboardModule as both an import and export in your app-extras.module.ts file.

Specifies the content to copy. This property binds data, so you must use brackets to evaluate the template expression: [copyTarget]. Without the brackets, Angular treats the value as a string and initializes the target property with the string value instead of evaluating the data value. Specifies a label for the copy to clipboard button. This property accepts string values. Specifies a label to display on the copy to clipboard button after users select it. This property accepts string values.
