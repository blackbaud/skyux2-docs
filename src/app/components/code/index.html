The code component formats text as a code snippet within larger blocks of text. Within the block of text, wrap sky-code tags around the code snippet. To format longer code snippets that stand alone as separate blocks of text, use the code block component instead.

If a code snippet has text within < and > characters, then the component automatically appends a closing tag unless the code snippet is valid HTML. To prevent this, use "&lt;" to escape < and "&gt;" to escape >. For example, escape the < and > characters in the code snippet public Task<Generic> MethodName() or the component automatically appends </Generic> to the end: public Task<Generic> MethodName()</Generic>.

To use the code component, you must set up the code block plugin. To install the plugin, run npm install @blackbaud/skyux-builder-plugin-code-block --save --save-exact. Then add the plugin to your skyuxconfig.json file: "plugins": ["@blackbaud/skyux-builder-plugin-code-block"].

Indicates whether to escape special characters such as {{ '{' }} and \ to render them as text instead of code.
