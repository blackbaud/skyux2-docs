The fluid grid component provides a responsive 12-column layout to organize content for all device sizes. The fluid grid consists of the sky-fluid-grid, sky-row, and sky-column tags. You place sky-column elements in sky-row elements, and then you can specify how much of the layout to use for each sky-column element on extra-small, small, medium, and large screens. The sky-fluid-grid parent element ensures consistent styling for all fluid grid instances. Indicates whether to reverse the display order for columns in the row. This property accepts boolean values. Specifies the number of columns (1-12) to use on extra-small screens (less than 768px). If you do not specify a value, the fluid grid displays the column at the full width of the screen. This property accepts number values. Specifies the number of columns (1-12) to use on small screens (768-991px). If you do not specify a value, the column inherits the screenXSmall value. This property accepts number values. Specifies the number of columns (1-12) to use on medium screens (992-1199px). If you do not specify a value, the column inherits the screenSmall value. This property accepts number values. Specifies the number of columns (1-12) to use on large screens (more than 1200px). If you do not specify a value, the column inherits the screenMedium value.
