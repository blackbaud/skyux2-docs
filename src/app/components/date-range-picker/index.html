The date range picker component creates a text input with a dropdown for users to select a date range from a set of well-known options. This includes options that enable users to specify dates to create their own date range. The text input is a composite component of up to two sky-datepicker components. Each calculator in the select has its own validator that is run when the date range value changes. The error is returned in the following format:

{ skyDateRange: { calculatorId: SkyDateRangeCalculatorId, errors: { [nameOfTheError: string]: boolean; } } }
Usage

Use the date range picker when users need to select a range of dates rather than a single date.

Behavior
Specific range

When users select "Specific range," from and to date fields appear.
Responsiveness

On smaller viewports, the date fields that appear when users select "Specific range" reflow into a one-column layout.
Layout
Error message

Display validation error messages below the date range picker.
