The date range picker component creates a text input with a dropdown for users to select a date range from a set of well-known options. This includes options that enable users to specify dates to create their own date range. The text input is a composite component of up to two sky-datepicker components. The options specify calculators to determine the dates in date ranges. This property accepts an array of SkyDateRangeCalculatorId values. By default, this property includes all SkyDateRangeCalculatorId values. Specifies a date format for the sky-datepicker components that make up the date range picker.
