The select field component creates a text input or button that launches a modal where users can select items. The modal displays a searchable, browsable, and filterable list of items. The component displays a text input to launch the modal when users can only select one item. It displays a button to launch the modal when users can select multiple items.

Usage

The select field component is effective for scenarios where users are not sure of the exact value they are looking for within a long list of items. For guidance on which selection inputs to use in other scenarios, see the form design guidelines.
