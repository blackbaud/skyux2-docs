The select field component creates a text input or button to launch a modal picker where users can select items from a searchable, browsable, and filterable list of items. When users can only select one item, the component displays a text input to launch the picker. When users can select multiple items, it displays a button instead.

Usage

The select field component is effective for scenarios where users are not sure of the exact value they are looking for within a long list of items. For guidance on which selection inputs to use in other scenarios, see the form design guidelines.
