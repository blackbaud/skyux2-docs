The select field component creates a text input or button to launch a modal where users can select items. The modal displays a searchable, browsable, and filterable list of items. When the component only allows users to select a single item, it displays a text input to launch the modal. When the component allows users to select multiple items, it displays a button to launch the modal.

Defines a data source to populate the select field component's modal with items that users can select. The SkySelectField type extends the any type and supports id, label, and category values for the data objects that you inject into the select field component. Defines a string value to label the select field for accessibility. If the select field includes a visible label on the screen, use the ariaLabelledBy property instead. Identifies the element that defines a label for the select field. If the select field does not include a visible label on the screen, use the ariaLabel property instead. Specifies an object property to highlight in the list of items that appear in the select field component's modal. For each item in the modal, the descriptorKey highlights the id, label, or category object property with bold text. Disables the select field to prevent users from opening the modal. Specifies whether users can select a single item or multiple items on the select field's modal. For single, the select field component displays a text field. For multiple, the component displays a button. Specifies the text to display on the select field button when the selectMode is multiple. Specifies the modal header. When selectMode is single, the default header is "Select a value." When selectMode is multiple, the default header is "Select values." Specifies the tooltip text for the icon to clear a selection when the selectMode is single. The clear icon appears in the select field after users select an item. Specifies the tooltip text for the select field when the selectMode is single. Specifies the ghost text to display in the select field appears when the selectMode is single and no item has been selected.
Usage

The select field component is effective for scenarios where users are not sure of the exact value they are looking for within a long list of items. For guidance on which selection inputs to use in other scenarios, see the form design guidelines.
