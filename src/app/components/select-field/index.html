The select field component launches a modal that displays items for users to select.

Defines a string value to label the select field for accessibility. If the select field includes a visible label on the screen, use the ariaLabelledBy property instead. Identifies the element that defines a label for the select field. If the select field does not include a visible label on the screen, use the ariaLabel property instead. Defines a data source to populate the select field component's modal with items that users can select. The SkySelectField type extends the any type and supports id, label, and category values for the data objects that you inject into the select field component. Specifies an object property to highlight in the list of items that appear in the select field component's modal. For each item on the modal, the descriptorKey of id, label, or category is formatted with bold text. Disables the select field to prevent users from opening the modal. Specifies whether users can select a single item or multiple items on the select field's modal. For single, the select field component displays a text field. For multiple, the component displays a button. Specifies the text to display on the select field button when the selectMode is multiple. . Specifies the tooltip text for the icon to clear a selection when the selectMode is single. The clear icon appears in the select field after users select an item. . Specifies the tooltip text for the select field when the selectMode is single. Specifies the ghost text to display in the select field appears when the selectMode is single and no item has been selected. Specifies the modal header. When the selectMode is single, the default value is Select a value. When the selectMode is multiple, the default value is Select values.
