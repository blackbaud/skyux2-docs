The toast module launches a container to display a message over a page's content. The SkyToastService provider launches toast messages and provides a common look and feel. The SkyToastMessage class triggers and watches for toast events. It also allows users to close messages.

The SkyToastService dynamically generates the toast component and appends it directly to the document's body element. Opens a message and displays the configured content. The config parameter is of type SkyToastConfig. This method returns the SkyToastMessage instance. Opens a message toast and displays the provided message. The message parameter is of type string and determines the message to be displayed. The optional config parameter is of type SkyToastConfig and adds additional configuration options to the toast. This method returns the SkyToastMessage instance. Opens a custom toast and renders the provided component in a toast. The customComponentType parameter is of type Type<SkyToastCustomComponent> and determines the component to be rendered. The optional config parameter is of type SkyToastConfig and adds additional configuration options to the toast. The optional providers parameter is of type Provider[] and specifies an array of custom providers to pass to the customComponentType's constructor. This method returns the SkyToastMessage instance.
