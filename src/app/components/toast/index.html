The toast module launches a container to display a message over a page's content. The SkyToastService provider lauches toasts and provides a common look-and-feel.

The SkyToastService launches toast messages and provides a common look and feel.

Opens a message toast and displays the specified message. The message parameter is of type string and determines the message to display. The optional config parameter is of type SkyToastConfig and adds additional configuration options to the toast. This method returns a SkyToastInstance instance. Opens a custom toast and displays the specified component. The component parameter determines the component to render. The optional config parameter is of type SkyToastConfig and adds additional configuration options to the toast. The optional providers parameter is of type StaticProvider[] and specifies an array of custom providers to pass to the component's constructor. This method returns a SkyToastInstance instance. Closes all open toasts.

Specifies the configuration options to set up a toast.

Specifies the type of toast and determines the color and icon to display. Available values are: SkyToastType.Info, SkyToastType.Success, SkyToastType.Warning, and SkyToastType.Danger.

An object that represents a single displayed toast.

Closes the toast. An observable that indicates when the toast is closed.
Usage

Toasts are appropriate when:

  • The trigger is server-side, indicating that a state in the system has changed.
  • The content is immediately important such that the user needs to know the information regardless of their current task or context in the system.
