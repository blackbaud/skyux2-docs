The form classes create styled inputs and labels for use in forms.

Form control

The sky-form-control class consistently styles an input and allows it to fill the width of its container.

Control label

The sky-control-label class consistently styles the label for an input. For guidance on the wording for labels, see the form design guidelines.

Required fields

The sky-control-label-required class indicates that an input is required to submit a form.

Error states

If a form field that uses Angular validation is in an invalid state, the input is styled with a red glow. The ng-invalid and ng-touched control status CSS classes indicate when the input is in an invalid state. For more information, see the Angular form validation docs.

In addition, the sky-error-label class provides styling for the error message when an input is in an invalid state. Place the message below the form field, and include logic to display the message when the conditions for invalid styling are met.
