The phone field module creates a button and text input to validate international phone numbers. The phone field button indicates the selected country and enables users to select different countries. Users enter phone numbers in the text input, and then the phone field input directive validates the numbers against standards for the selected country. For validation, phone numbers are driven through the ngModel attribute that you specify on an input element or on a FormControl in a reactive form.

Use the phone field to validate the phone number the user enters against the correct format for the selected country.

Default country

By default, the country selector will be set to the United States. You can specify a different default value or set the default dynamically, for example by using location detection.

Country selection

If the user is entering a phone number from a country other than that currently selected, they must choose the correct country to ensure the phone number is validated against the expected format.

Phone format validation

The phone input is validated against the format expected for the selected country. You have to specify the error message; in most cases the error message should say "Enter a phone number matching the format for the selected country."
