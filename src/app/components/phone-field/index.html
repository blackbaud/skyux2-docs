The phone field module creates a button and text input to validate international phone numbers. The phone field button indicates the selected country and enables users to select different countries. Users enter phone numbers in the text input, and then the phone field input directive validates the numbers against standards for the selected country. For validation, phone numbers are driven through the ngModel attribute that you specify on an input element or on a FormControl in a reactive form.
