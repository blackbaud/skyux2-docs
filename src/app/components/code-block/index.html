The code block component formats a block of text as a code snippet with syntax highlighting. The code block also has a header that displays a copy button and, if defined, the language type. To create a code snippet, wrap sky-code-block tags around the code block.

If a code block has text within < and > characters, then the component automatically appends a closing tag unless the code snippet is valid HTML. To prevent this, you must use "<" to escape < and ">" to escape >. For example, you must escape the < and > characters in the code snippet public Task<Generic> MethodName() or else the component automatically appends </Generic> to the end: public Task<Generic> MethodName()</Generic>.

Using the list below, see if your code snippet language is supported. The languages supported appear in pairs. The first part lists the language and the second part displays the text you use for the languageType property. For example, if your code block is a JavaScript code snippet, use languageType="javascript".
