The code block component formats a block of text as a code block with syntax highlighting. The code block's header can display a copy button and language type. To create a code block, wrap sky-code-block tags around the text.

If a code block has text within < and > characters, then the component automatically appends a closing tag unless the code snippet is valid HTML. To prevent this, use "&lt;" to escape < and "&gt;" to escape >. For example, escape the < and > characters in the code snippet public Task<Generic> MethodName() or the component automatically appends </Generic> to the end: public Task<Generic> MethodName()</Generic>.

To use the code block component, you must set up the code block plugin. To install the plugin, run npm install @blackbaud/skyux-builder-plugin-code-block --save --save-exact. Then add the plugin to your skyuxconfig.json file: "plugins": ["@blackbaud/skyux-builder-plugin-code-block"].
