The code block component formats a block of text as a code block with syntax highlighting. The code block's header can display a copy button and language type. To create a code block, wrap sky-code-block tags around the code block.

If a code block has text within < and > characters, then the component automatically appends a closing tag unless the code snippet is valid HTML. To prevent this, use "&lt;" to escape < and "&gt;" to escape >. For example, escape the < and > characters in the code snippet public Task<Generic> MethodName() or the component automatically appends </Generic> to the end: public Task<Generic> MethodName()</Generic>.

To use the code block component, set it up as a dependency in your SKY UX SPA. First, install the package as part of your SPA dependencies: npm install @blackbaud/skyux-lib-code-block --save. Then install its plugin: npm install @blackbaud/skyux-builder-plugin-code-block --save. Next, add SkyCodeBlockModule as both an import and export in your app-extras.module.ts file. Finally, add the code block plugin to your skyuxconfig.json file: "plugins": ["@blackbaud/skyux-builder-plugin-code-block"].
