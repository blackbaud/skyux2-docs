The label component provides a compact, eye catching design element to draw attention to status information that users need to address. Statuses indicate a state or condition at a particular time and fall into four categories: info, success, warning, and danger. The type of label to display. Possible values are info, success, warning and danger.

Some statuses are crucial to user tasks, while others are no more important than other information on a page. If statuses are crucial to user tasks, use labels. If not, use regular text.

Label placement

Use labels in a summary context. For example, you can place a label in a page summary. To convey statuses inline to indicate that they are tied to specific pieces of data, use status indicators instead.
