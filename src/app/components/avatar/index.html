The avatar component displays an image to identify a record and includes an option to let users change the image. If the image is missing, the avatar component can display initials for the record that the avatar represents.

Indicates whether users can change the image. To select a different image, users click the image or drag another image on top of it, much like the sky-file-drop component in the file attachment module. This property accepts boolean values. Specifies the name of the record that the avatar represents. If the src property does not specify an image, the component extracts initials from the first and last words of this name and displays them in place of the missing image. To ensure that the component extracts the correct initials, specify a name with no prefix or suffix. You can specify just the initials with a space between them. This property accepts string values. Specifies an image to display to identify a record. This property accepts string, Blob, or File values. Emits a SkyFileItem object when the image is updated. Specifies error details if the file causes an error. Specifies an error type if the file causes an error. Specifies the File object associated with the SkyFileItem object. Specifies the URL of the file item.
