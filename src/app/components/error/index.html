The error component displays a SKY UX-themed error message. It includes options to specify the image, title, description, and action. Specifies an image to display in the error message. Specifies a title to display in the error message. Specifies a description to provide additional details in the error message. Specifies an action to include in the error message. For example, you can include a button to reload the page or to refresh data. An attribute to specify a set of pre-defined values for the image, title, and description. The valid options are broken, notfound, construction, and security. Opens a modal to display a SKY UX-themed error message. The config object specifies the text to display in the error message. Specifies a title to display in the modal error message. Specifies a description to provide additional details in the modal error message. Specifies a label for the action button that closes the modal error message.
