The sky-tile component creates a collapsible container that serves as a building block for pages and forms in SKY UX applications. Within the sky-tile component, the sky-tile-title component displays a header for the tile, the sky-tile-summary component displays summary information, and the sky-tile-content component displays the tile content. The sky-tile-content-section component creates a padded section inside a sky-tile-content element. The sky-tile-dashboard component creates a container to group multiple tiles. To display a sky-tile component within a sky-tile-dashboard component, you must include it as an entry component in the application module.

The selector for a tile component to display in a tile dashboard must begin with div. followed by a custom selector.

Tiles are containers to organize varied content. They divide pages into multiple categories and group related content. Use tiles to provide access to a wide range of content where users benefit from the ability to reorder, expand and collapse, and personalize tiles.

Usage

Use tiles to divide pages with many categories of content, especially when the content is varied enough that users would benefit from the ability to reorder or expand/collapse the categories.

Don't use tiles to divide pages into fewer than 5 sections. However, if tile content is complex enough for different users to repeatedly and reliably prioritize fewer tiles, you can divide pages into as few as 3 tiles.

Don't use a single tile to contain all of a page's content after the summary.

Don't manipulate the widths of tiles to create asymmetrical layouts. Uniform widths preserve the ability of users to reorder, expand, and collapse tiles.

Options
Help

For tasks where users require documentation for a tile, invoke help from the tile header. If only a few items in the tile need explanation, use element-level help instead. In general, don't combine tile-level help and element-level help on items within the tile.

For tiles that are not populated, use empty-state help to explain how to use the tile.
