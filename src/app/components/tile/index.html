The sky-tile component creates a collapsible container that serves as a building block for pages and forms in SKY UX applications. Within the sky-tile component, the sky-tile-title component displays a header for the tile, the sky-tile-summary component displays summary information, and the sky-tile-content component displays the tile content. The sky-tile-content-section component creates a padded section inside the sky-tile-content element. If you use a sky-tile component within a sky-tile-dashboard component, you must include it as an entry component in the application module.

The UI Config Service applies sticky settings to tile dashboards and saves configuration settings for users. This property accepts string values. This property is for internal Blackbaud use only. For Blackbaud developers, the UI Config Service website describes how to apply sticky settings to tile dashboards. Fires when the tile dashboard changes state and emits a SkyTileDashboardConfig object. This occurs when tiles collapse or expand and when users drag and drop tiles to rearrange them. An array of SkyTileDashboardConfigTile objects that specifies the tiles to include in the dashboard. Each SkyTileDashboardConfigTile object includes the following properties:
  • id — The unique identifier for the tile.
  • componentType — The class type for the tile component.
  • providers — (Optional.) An array of data providers that can be passed to the tile.
A SkyTileDashboardConfigLayout object that describes the tile dashboard's layout. The SkyTileDashboardConfigLayout object contains a singleColumn property of type SkyTileDashboardConfigLayoutColumn and a multiColumn property that is an array of type SkyTileDashboardConfigLayoutColumn. The singleColumn layout displays on small screens, while the multiColumn layout displays on larger screens. The SkyTileDashboardConfigLayoutColumn object contains an array of SkyTileDashboardConfigLayoutTile objects that indicates the order of tiles. It includes the following properties:
  • id — The identifier for the tile in the dashboard.
  • isCollapsed — A boolean value that indicates whether the tile is in a collapsed state.
Fires when the tile dashboard's initialization is complete.
