The sky-tile component creates a collapsible container that serves as a building block for pages and forms in SKY UX applications. Within the sky-tile component, the sky-tile-title component displays a header for the tile, the sky-tile-summary component displays summary information, and the sky-tile-content component displays the tile content. The sky-tile-content-section component creates a padded section inside a sky-tile-content element. The sky-tile-dashboard component creates a container to group multiple tiles. To display a sky-tile component within a sky-tile-dashboard component, you must include it as an entry component in the application module.

A SkyTileDashboardConfigLayout object that describes the tile dashboard's layout. It includes the following properties:
  • singleColumn — A SkyTileDashboardConfigLayoutColumn object that describes how to display tiles in a single column on small screens.
  • multiColumn — An array of SkyTileDashboardConfigLayoutColumn objects that describes how to display tiles in multiple columns on larger screens.

    • SkyTileDashboardConfigLayoutColumn objects include the following property:

    • tiles — An array of SkyTileDashboardConfigLayoutTile objects that describe the tiles to include in the columns of the dashboard and the order to display them. It includes the following properties:
      • id — Specifies the ID of a tile to display in the dashboard. This property accepts string values.
      • isCollapsed — Indicates whether the tile is in a collapsed state. This property accepts string values.
An array of SkyTileDashboardConfigTile objects that specifies the tiles to include in the dashboard. Each SkyTileDashboardConfigTile object includes the following properties:
  • componentType — Specifies the class type of the tile component.
  • id — Specifies the ID of the tile. This property accepts string values.
  • providers — (Optional.) Specifies an array of data providers that can be passed to the tile. This property accepts an array of any values.
A SkyTileDashboardConfigReorderData object that describes how to move a tile within the dashboard. It includes the following properties:
  • column — Specifies the column for the tile. This property accepts number values.
  • position — Specifies the position of the tile within the column. This property accepts number values.
  • tileDescription — Specifies the description of the tile. This property accepts string values.
