The sky-tile directive creates a collapsible container that is a building block for pages and forms in SKY UX applications. The sky-tile-section directive creates a padded section inside a sky-tile element. The sky-tile-summary directive displays summary information inside a sky-tile element. If you use a sky-tile directive within a sky-tile-dashboard directive, you must include it as an entry component in the application module.

Indicates whether the tile is in a collapsed state. Event emitted when the settings button on the tile is clicked. The settings button only appears if this output is placed on the tile. Event emitted when the tile changes collapsed state. Returns true if the tile is collapsed. Otherwise, returns false. The SkyTileDashboardConfig object that populates the tile dashboard. This event emits a SkyTileDashboardConfig object when the tile dashboard changes state. This occurs when tiles are collapsed or expanded and when tiles are rearranged via drag and drop. An array of SkyTileDashboardConfigTile objects that indicates which tiles are contained in the dashboard. Each SkyTileDashboardConfigTile has the following properties:
  • id — The unique identifier for the tile.
  • componentType — The class type for the tile component.
  • providers — An array of data providers that can be passed to the tile.
A SkyTileDashboardConfigLayout object that describes the layout of the tile dashboard. The SkyTileDashboardConfigLayout object contains a singleColumn property of type SkyTileDashboardConfigLayoutColumn and a multiColumn property that is an array of type SkyTileDashboardConfigLayoutColumn. The singleColumn layout displays on small screens, while the multiColumn layout displays on larger screens. The SkyTileDashboardConfigLayoutColumn object contains an array of SkyTileDashboardConfigLayoutTile objects that indicates the order of tiles. It has the following properties:
  • id — The identifier for the tile in the dashboard.
  • isCollapsed — A boolean value that indicates whether the tile is in an expanded or collapsed state.
