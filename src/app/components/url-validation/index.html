The URL validation module validates the format of URLs in input fields. Creates an input that validates URLs. Place this attribute on an input element. If users enters values that are not valid URLs, an error message appears. This directive uses NgModel to bind data. Validates URLs in reactive forms. Add this validator directly to the form control model in the component class. If users enter values that are not valid URLs, the validator throws an error. Since this is a sync validator, it returns a set of validation errors or undefined immediately when users enter values.
