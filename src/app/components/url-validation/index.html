The URL validation module allows the user to validate URL format. When this attribute is placed on an input element, the input with validation properties is created. The error message will display if the input is not in the correct format. This directive will use NgModel for binding data. Creates an input that validates a url in a reactive form. Add this validator directly to the form control model in the component class. If users enters a value that is not a valid url, an error message appears. Since this is a sync validator, it will return a set of validation errors or undefined immediately upon entering a value.
