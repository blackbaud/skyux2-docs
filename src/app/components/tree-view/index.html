Tree views display a hierarchical view of items and have multiple modes for selecting items.

This component is a release candidate. We do not plan to make any breaking changes the component before the official release, but such changes are possible.

Use when

Use tree views to show lists of hierarchical items. They can be used in view-only or selection contexts.
Do use tree views to show lists of hierarchical items.
 Expand/collapse button List item content Checkbox Context menu

Selection

You can use single- or multi-select mode when you use a tree view in a selection context.

With cascading selection when the user selects a node, all descendant nodes are also selected. Use this mode if it's likely that users will want to select a node and all its descendants.

With leaf-only selection the user can only select the leaf nodes in the tree. Use the leaf-node-only selection mode if the higher-level nodes in the tree are categories and you want the user to only select the leaf node items.

Split view

You can use tree views as the list in a split view if the user needs to work with hierarchical items.

Expand/collapse nodes

Clicking the expand/collapse button expands/collapses the node.
