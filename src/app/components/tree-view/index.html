The tree view provides a hierarchical view for a list of items.

Use when

Use tree views to show lists of hierarchical items. They can be used in view-only or selection contexts.
Do use tree views to show lists of hierarchical items.
 List item Expand/collapse button Checkbox Context menu

Selection

You can use single- or multi-select mode when you use a tree view in a selection context.

Use the cascading selection mode if it's likely that users will want to select a node and all nodes below it in the hierarchy.

Use the leaf-node-only selection mode if the higher-level nodes in the tree are categories and you want the user to only select the leaf node items.

Split view

You can use tree views as the list in a split view if the user needs to work with hierarchical items.

Expand/collapse nodes

Clicking the expand/collapse button expands/collapses the descendant items of the node.
