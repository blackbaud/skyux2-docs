This list view grid component displays a grid for a SKY UX-themed list of data. Internally, the sky-list-view-grid uses the sky-grid component, which contains documentation for sky-grid-column.

The list view grid component is designed for use within a list component.

Specifies columns to display by default. Will match off of the ID or field of the item. Acceptable values: Array of strings or Observable Array of strings. Specifies how the grid should fit to its parent. Acceptable values: 'width' or 'scroll'. Specifies the height of the grid. Acceptable values: number or Observable number. Specifies the columns to hide by default. Will match off of the ID or field of the item. Acceptable values: Array of strings or Observable Array of strings. Specifies the name of the view. Acceptable values: string. Specifies the id of the row to highlight. Will match the id property of the data object. Typically used in conjunction with the flyout component to indicate which row is currently selected. Acceptable values: string. Specify a search function to apply on the view data. Acceptable value: (data: any, searchText: string) => boolean. Specifies the width of the grid. Acceptable values: number or Observable number. Fires when the columns change. This includes changes to the columns to display in the list view grid and changes to the order of columns. The event emits an array of IDs for the displayed columns that reflects the column order. The sky-list-column-selector-action component can be used within the list toolbar component to provide a column selector modal for a grid view. Set the gridView property of the sky-list-column-selector-action to enable the column selector.
