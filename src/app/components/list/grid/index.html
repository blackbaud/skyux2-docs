The list view grid component displays a grid for a SKY UX-themed list of data. It is designed for use within a list component. Internally, the sky-list-view-grid uses the sky-grid component, which contains documentation for sky-grid-column.

To use the list view grid component, you must install SkyListModule as a dependency to your project. For information about SkyListModule, see List overview.

Specifies the name of the view. This property accepts string values. Specifies the columns to display by default based on the ID or field of the item. This property accepts an array or Observable array of string values. Indicates whether to enable the multiselect feature to display a column of checkboxes on the left side of the grid. Multiselect also displays an action bar with buttons to select and clear all checkboxes. Multiselect defaults to the id property on the list's data object. This property accepts boolean values. Specifies how the grid fits to its parent. The valid options are width, which fits the grid to the parent's full width, and scroll, which allows the grid to exceed the parent's width. If the grid does not have enough columns to fill the parent's width, it always stretches to the parent's full width. This property accepts string values. Specifies the height of the grid. This property accepts number or Observable number values. Specifies the columns to hide by default based on the ID or field of the item. This property accepts an array or Observable array of string values. Indicates whether to highlight search text within the grid. This property accepts boolean values. Specifies the ID of the row to highlight. The ID matches the id property of the data object. Typically, this property is used in conjunction with the flyout component to indicate the currently selected row. This property accepts string values. Specifies a search function to apply on the view data. This property accepts values of type (data: any, searchText: string) => boolean. Specifies a unique key for the UI Config Service to retrieve stored settings from a database. The UI Config Service saves configuration settings for users and returns selectedColumnIds to preserve the columns to display and the preferred column order. This property accepts string values. When the multiselect feature is enabled and the selected checkboxes change, the list component's selectedIdsChange event emits a Map string, boolean object to indicate the selected items. This differs from multiselect behavior in the grid component where the multiselectSelectionChange event emits an array of IDs for the selected rows when the selection changes.

Fires when columns change. This includes changes to hide or display columns and changes to the order of columns. The event emits an array of IDs for the displayed columns that reflects the column order.

The sky-list-column-selector-action component provides a column selector modal for a list grid view when placed in a sky-list-toolbar component. For information the toolbar component, see the list toolbar.
