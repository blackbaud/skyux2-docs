The list view grid component displays a grid for a SKY UX-themed list of data. It is designed for use within a list component. Internally, the sky-list-view-grid uses the sky-grid component, which contains documentation for sky-grid-column.

Specifies the name of the view. This property accepts string values. Specifies the columns to display by default based on the ID or field of the item. This property accepts an array or Observable array of string values. Specifies how the grid fits to its parent. The valid options are width, which fits the grid to the parent's full width, and scroll, which allows the grid to exceed the parent's width. If the grid does not have enough columns to fill the parent's width, it always stretches to the parent's full width. This property accepts string values. Specifies the height of the grid. Typically, this property is used in conjunction with the flyout component to indicate the currently selected row. This property accepts string values. Specifies a search function to apply on the view data. This property accepts of type (data: any, searchText: string) => boolean. Specifies the width of the grid. This property accepts number or Observable number values. Fires when columns change. This includes changes to hide or display columns and changes to the order of columns. The event emits an array of IDs for the displayed columns that reflects the column order. The sky-list-column-selector-action component is available in the list toolbar component to provide a column selector modal for a grid view. To enable the column selector, set the gridView property of the sky-list-column-selector-action.
