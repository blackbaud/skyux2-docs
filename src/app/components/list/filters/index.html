The list filters module provides filtering capabilities for a SKY UX-themed list of data. It is designed for use within a list component.

Contains a filter button for the list toolbar. Place a sky-filter-button component inside this component to open a modal with filtering options. To apply filter options, use the appliedFilters property on the sky-list component. Creates an inline filter area for the list. Place each filter in a sky-list-filter-inline-item component. Creates a filter summary based on the list's appliedFilters property. Place this component within the sky-list-toolbar component. Specifies the name of the filter. This property accepts string values. Specifies the template for the filter. The template can access the filter variable that contains the value of the filter control, which should be bound to ngModel, and the changed function, which should be called when the model changes. Specifies the default value of the filter. If the value of the filter is set to the default value, then the filter is not applied. Specifies the function to apply to determine whether an item is filtered. This property is required when using in memory data provider. Acceptable values: (item: ListItemModel, filterValue: any) => boolean. Specifies the initial value of the filter. Emits a ListFilterModel when users click a summary item. Acceptable values: (item: ListItemModel, filterValue: any) => boolean. Specifies the text for the filter's summary item.
