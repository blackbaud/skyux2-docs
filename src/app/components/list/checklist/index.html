The list view checklist component displays a SKY UX-themed checklist of data. It includes the ability to defined a custom search function for the data given.

The list view checklist component is designed for use within a list component.

When set to single, the checklist only allows selection of one item in the checklist. Specifies the name of the view. Acceptable values: string. Specifies the name of the label field selector. Acceptable values: string. Specifies the name of the description field selector. Acceptable values: string. Specifies a search function to apply on the view data. Default searches view data on label, description, and category if field selectors are defined for the given field. Acceptable value: (data: any, searchText: string) => boolean.
