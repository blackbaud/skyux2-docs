The list view checklist component displays a checklist within a SKY UX-themed list of data and includes the ability to define a custom search function for the data.

Specifies the name of the view. Acceptable values: string. Specifies the name of the description field selector. Acceptable values: string. Specifies the name of the label field selector. Acceptable values: string. Specifies a search function to apply on the view data. The default function searches view data on label, description, and category if field selectors are defined for the given field. Acceptable value: (data: any, searchText: string) => boolean. Specifies whether users can select one item or multiple items in the checklist. single allows users to select one item in the checklist, while multiple allows users to select multiple items in the checklist. Indicates whether to display selected items only.
