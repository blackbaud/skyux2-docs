The dropdown component renders a button that displays a menu when users click it. The menu displays a set of actions or functions. Do not use the dropdown component as a select input for users to set the menu's value to one of the options. If you need a select input, see the options to select items from lists.

Specifies the type of button to render as the dropdown's trigger element. To display a button with text and a caret, specify select and then enter the button text in a sky-dropdown-button element. To display a round button with an ellipsis, specify context-menu. And to display a button with a Font Awesome icon, specify the icon's class name. For example, to display the fa-filter icon, specify filter. Specifies a background color for the dropdown button. If unspecified, the default value sets the background color to sky-color-white. To change the background color to sky-color-blue, specify primary. Specifies an accessibility label to provide a text equivalent for screen readers when the dropdown button has no text. Specifies a title to display in a tooltip when users hover the mouse over the dropdown button. Provides an observable to send commands to the dropdown. The commands should respect the SkyDropdownMessage type (see below). Indicates whether to use the browser's native focus function when users navigate through menu items with the keyboard. To disable the native focus function, set this property to false. For example, if you want to interact with the dropdown menu but keep the keyboard focus on a different element, set this property to false. Provides an observable to emit changes when the menu's active index or selected item changes. The response is of type SkyDropdownMenuChange (see below). Sets the message to one of the following SkyDropdownMessage types:
  • SkyDropdownMessageType.Open — Opens the dropdown menu.
  • SkyDropdownMessageType.Close — Closes the dropdown menu.
  • SkyDropdownMessageType.FocusTriggerButton — Puts focus the dropdown button.
  • SkyDropdownMessageType.FocusNextItem — Puts focus on the next item in the dropdown menu.
  • SkyDropdownMessageType.FocusPreviousItem — Puts focus on the previous item in the dropdown menu.
  • SkyDropdownMessageType.Reposition — Repositions the dropdown next to the dropdown button. This is useful for when the dropdown's width and height change while open.
Indicates the active menu index. Indicates the selected item in the menu. The value of the field is of type SkyDropdownItemComponent.
