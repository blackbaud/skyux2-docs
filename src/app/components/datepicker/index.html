The datepicker module contains directives to let users select dates or fuzzy dates. It is a SKY UX-themed replacement for the HTML input element with type="date". The value that users select is driven through the ngModel attribute specified on the input element.

To use the datepicker module, you must install SkyPopoverModule as a dependency to your project. For more information, see the popover module.
