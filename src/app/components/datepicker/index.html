The datepicker module contains directives to let users select dates. It is a SKY UX-themed replacement for the HTML input element with type="date". The value that users select is driven through the ngModel attribute specified on the input element. Creates the datepicker input and calendar. Place this directive on an input element, and wrap the input in a sky-datepicker component. Specifies the date format for the input. Place this attribute on the input element to override the default date format in SkyDatepickerConfigService. This property accepts string values. Indicates whether to disable the datepicker. This property accepts boolean values. Specifies the maximum selectable date in the calendar. Place this attribute on the input element to override the default maximum in SkyDatepickerConfigService. This property accepts Date values. Specifies the minimum selectable date in the calendar. Place this attribute on the input element to override the default minimum in SkyDatepickerConfigService. This property accepts Date values. Indicates whether to prevent validation on the datepicker input. When set to true, date validation does not run on the input. This property accepts boolean values. Specifies the starting day of the week in the calendar, where 0 sets the starting day to Sunday.
