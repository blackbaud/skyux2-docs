The single file attachment component creates an input to attach a single local file.

Use when

Use the single file attachment input when users need to upload a single local file.
Do use the single file attachment input to upload one file.

Don't use when

Don't use the single file attachment input when users need to link to a URL to attach an external file. Use the file attachment component instead.

Don't use the single file attachment input to upload multiple local files. Use the file attachment component instead.
Don't use single file attachment inputs to upload more than one local file.

The file attachment component provides an input to browse for a single local file or to use as a drop zone to drag and drop a file. After users attach a file, the component displays a link that emits an event when selected, and it also displays a preview for image files.

Specifies a comma-delimited string literal of MIME types that users can attach: [acceptedTypes]="validFileTypes" or acceptedTypes="image/png,image/jpeg". By default, all file types are allowed.
