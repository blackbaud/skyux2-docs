The vertical tabs module contains components to render a vertical tabset that display large amounts of content without extending past the edge of the screen. It also allows you to group related tabs into collapsible sections. The sky-vertical-tabset component creates a container for the vertical tabs, and the sky-vertical-tabset-group component creates a collapsible container to organize related tabs into a section. Within the sky-vertical-tabset and sky-vertical-tabset-group components, the sky-vertical-tab component creates an individual tab.

When to use vertical tabs
  • Use vertical tabs when a page has large amounts of content to divide into separate pages and standard tabs would extend past the edge of the screen.
  • Use vertical tabs when it is important to group related tabs into collapsible sections.
When not to use vertical tabs
  • The standard tab control is the preferred option to divide large subsets of content on a page. Do not use vertical tabs when the standard tab control will work.
  • Do not use more than one set of vertical tabs on a page.
  • Do not use vertical tabs on a form. Use the sectioned form component instead.
