The popover component allows you to display contextual content when users select a button or link. The content can be plain text, HTML, or any Angular component. To close a popover, users can select the trigger element or click elsewhere on the page. References a SkyPopoverComponent instance on the page. Available values are left, right, and center. Specifies the placement of the popover in relation to the trigger element. Available values are above, below, left, and right. Fires when users close the popover.
Usage

Popovers are appropriate when:

  • The trigger is client-side, a direct result of a user interaction with an element (such as a click) that does not lead to a change in system state.
  • The content is directly related to the user's current task.
  • The content will not require scrolling on a mobile device.

If the content is relevant most of the time, present it in-page instead. If the amount of content would require scrolling on a mobile device, use a flyout instead.

Interaction

To ensure usability on touch devices, you should generally trigger popovers on click rather than mouseenter.
