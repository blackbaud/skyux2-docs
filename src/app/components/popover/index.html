The popover component allows you to display contextual content when users select a button or link. The content can be plain text, HTML, or any Angular component. To close a popover, users can select the trigger element or click elsewhere on the page. References a SkyPopoverComponent instance on the page. Add this directive to an element that will open a popover when selected. Specifies the horizontal alignment of the popover in relation to the trigger element. Available values are left, right, and center. Specifies the placement of the popover in relation to the trigger element. Available values are above, below, left, and right. Specifies a user action that displays the popover. Available values are click and mouseenter. Specifies a title for the popover. Specifies the default placement value of the popover. Can be overwritten by the skyPopoverPlacement property. Specifies the default alignment value of the popover. Can be overwritten by the skyPopoverAlignment property. Specifies the template reference variable that the [skyPopover] directive uses. Indicates whether to close the popover when focus is lost. To require users to click a trigger button to toggle the popover's visibility, set this input to false. By default, the popover closes when focus is lost. Fires when users open the popover. Fires when users close the popover.

To ensure usability on touch devices, you should generally trigger popovers on click rather than mouseenter.
