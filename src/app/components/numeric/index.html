The numeric pipe formats numbers to shorten values of 1000 or more to rounded numbers and abbreviation characters such as K for thousands, M for millions, B for billions, and T for trillion. The pipe can also format shortened number as currency, with proper localization and currency symbol placement.

Pipe usage: number_expression | skyNumeric[:NumericOptions] ]]

When you create a NumericOptions object in the template, you MUST have the space between the closing bracket for the object and the closing brackets for the pipe or it won't work. For examples, see the HTML code sample below.

You can also import SkyNumericService and call the formatNumber method. The method uses a NumericOptions class that includes arguments to format the number. You create a new instance of the class with its parameters set and pass it in as the NumericOptions parameter.

Method usage: this.skyNumeric.formatNumber(value, NumericOptions)

Specifies the number to format. This property accepts number values. Specifies the NumericOptions object that provides arguments for the number to format. Specifies the maximum number of digits after the decimal separator. This property accepts number values. Specifies how to format the number. Options are currency or number. This property accepts string values. Specifies the ISO4217 currency code to use for currency formatting. If you do not specify a currency code, the component uses the browser's culture to determine the currency unless your SPA provides a different culture with SkyAppLocaleProvider. This property accepts string values. Specifies the minimum number of digits after the decimal separator. This property only applies when the truncate property is set to false. If digits specifies a maximum number of digits, then minDigits must be less than that value. This property accepts number values. Indicates whether to shorten numbers to rounded numbers and abbreviation characters such as K for thousands, M for millions, B for billions, and T for trillion. This property accepts boolean values. Specifies the starting point after which numbers are shortened to rounded numbers and abbreviation characters. This property accepts number values.
