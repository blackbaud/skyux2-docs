The tabs module contains components to render tabs that divide large subsets of content on a page. The module includes options to display buttons in the tab area and to make tabs closeable. The sky-tabset component creates a container for the tabs, and within it, the sky-tab component creates an individual tab. The content within a sky-tab component is displayed when that tab is active.
