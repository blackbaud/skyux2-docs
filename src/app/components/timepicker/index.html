The timepicker module contains directives to let users select times. It is a SKY UX-themed replacement for the HTML input element with type="time". The value that users select is driven through the ngModel attribute specified on the input element. Creates the timepicker input and dropdown. Place this attribute on an input element, and wrap the input in a sky-timepicker component. This attribute must be set to the instance of the sky-timepicker. Indicates whether to disable the timepicker. This property accepts boolean values. Specifies a custom time format. See the moment.js docs for examples. This property accepts string values. Specifies the 12-hour hh or 24-hour HH time format for the input. This property accepts string values. Fires when the value in the timepicker input changes.
