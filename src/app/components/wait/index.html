The wait component and service allow you to create waits on individual elements as well as on a page.

When set to true, wait indication appears on the parent element of the sky-wait component. When set to true, wait indication appears in the bottom left corner of the element instead of hiding the entire parent element. When set to true, wait indication appears on the page level instead of the parent element level. We recommend that you use the beginBlockingPageWait or beginNonBlockingPageWait functions of the SkyWaitService instead of setting this on the component level. Defines a string value to label the wait icon for accessibility while an element or page loads. Starts a blocking page wait on the page. Ends a blocking page wait on the page. Blocking page wait indication is removed when all running blocking page waits are ended. Starts a non-blocking page wait on the page. Ends a non-blocking page wait on the page. Non-blocking page wait indication is removed when all running non-blocking page waits are ended. Clears all blocking and non-blocking page waits on the page.
