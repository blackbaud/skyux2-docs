The search component creates an input control for users to enter search criteria. Specifies how many milliseconds to wait before searching after users enter text in the search input. Indicates whether to disable the filter button. Specifies the expand mode for the search input. The valid options include responsive to collapse the search input into a button on mobile devices, none to not collapse the search input on mobile devices, and fit to extend the search input to fit the width of its container. Specifies placeholder text to display in the search input until users enter search criteria. Specifies default search criteria for the input. Fires when the search text is applied. Fires when the search text is changed. Fires when the search text is cleared.
