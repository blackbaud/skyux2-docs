The definition list module displays a list of label-value pairs. Creates a definition list to display label-value pairs. Wraps the label-value pairs in the definition list. Specifies the label in a label-value pair. Specifies the value in a label-value pair. Specifies a title for the definition list. Specifies the width of the label portion of the definition list. This property accepts string values. Specifies the text to display in the value portion of the definition list when no value is available in a label-value pair. This property accepts string values.

Don't append colons to definition list labels.
